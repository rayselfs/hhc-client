# Frontend Framework

- **UI Framework**: **Vuetify 3** is the primary UI library.
  - Use standard Vuetify components (`v-container`, `v-row`, `v-col`) for layouts.
  - Follow Material Design 3 principles provided by Vuetify.
  - Prefer using Vuetify's built-in Directives (e.g., `v-ripple`, `v-click-outside`).

- **Styling**: Avoid custom CSS when possible; use Vuetify's Utility Classes (e.g., `ma-4`, `pa-2`, `text-h5`, `d-flex`).
- **Locale**: Use vue-i18n
- **API**: Use axios
- **font**: Use @fontsource-variable
